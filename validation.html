Les requêtes en XPath

Voici nos requêtes :

  1. pour les documents XML respectant la DTD en 7 : fournir les contenus des éléments glossaire :

    //descendant::glossaire/child::mots/child::text()

    Vous pouvez la tester ici :Test requête A en Xpath

  2. pour les documents XML respectant la DTD en 7 : fournir le premier élément glossaire :

    /descendant::glossaire[1]

    Vous pouvez la tester ici :Test requête B en Xpath

  3. pour les documents XML respectant la DTD en 7 : fournir les éléments n’ayant pas d’attribut auteur:

    /foot/glossaire[not(@auteur)]

    Vous pouvez la tester ici :Test requête C en Xpath

  4. pour vos documents XHTML : fournir les valeurs des attributs href :

    //a/@href

    Vous pouvez la tester ici :Test requête D en Xpath

  5. pour vos documents XHTML : compter le nombre d'images dans un document XHTML:

    count(//img)

    Vous pouvez la tester ici :Test requête E en Xpath
